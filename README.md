The Weather Dashboard App is a modern and user-friendly application that provides real-time weather information using the OpenWeatherMap API. Users can search for any city worldwide and instantly view current weather conditions, including temperature, humidity, wind speed, atmospheric pressure, and visibility. The app also features hourly and multi-day weather forecasts, helping users plan their activities effectively.

Built with a clean and responsive interface, the dashboard offers an intuitive user experience across desktop and mobile devices. The project demonstrates API integration, data visualization, responsive design, and dynamic content rendering, making it a practical solution for accessing accurate weather information in real time.

Key Features:

Real-time weather updates
City-based weather search
Hourly and 5-day forecasts
Temperature, humidity, wind speed, and pressure details
Responsive and modern UI
OpenWeatherMap API integration
Fast and accurate weather data visualization
